2. Основные шаги:
   - Измените DEBUG = False в settings.py
   - Установите SECRET_KEY на безопасный ключ
   - Используйте PostgreSQL вместо SQLite
   - Настройте ALLOWED_HOSTS на ваш домен
   - Используйте Gunicorn + Nginx

3. SSL сертификат:
   - Используйте Let's Encrypt (certbot)
   - Включите SECURE_SSL_REDIRECT = True
